hostapd-statistics
==================

Statistics for Hostapd


Needs: socat, nmap, inotifywait, arp-scan

Optional: vnstati, sensors, iw


Works with logrotate

This project uses some of the Tango Desktop Project icons. http://tango.freedesktop.org/
